# Release

This directory stores the final released dataset generated by the codebase.

If you run the full construction workflow, the resulting dataset should be saved here.

## Output

The released dataset is provided in **Parquet** format.

It includes the core components of the municipal multi-regional input-output (MRIO) table for Korea, 2020:

- domestic intermediate transactions
- imports
- final demand
- value added
- total output

## Published dataset

The dataset has been released on Zenodo:

**DOI:** https://doi.org/10.5281/zenodo.19217463 

## Notes

This folder is intended for the final released output only.
Intermediate files and processing scripts are managed elsewhere in the repository.
